# Quotes Recommendation Chatbot (NLP)

## Overview
This project implements an NLP-based chatbot that recommends quotes 
based on user intent using TF-IDF and Logistic Regression.

## Features
- Intent Classification
- Machine Learning Model
- Model Persistence
- Modular Code Structure

## Tech Stack
- Python
- scikit-learn
- pandas
- nltk

## How to Run

1. Train Model:
   python src/train.py

2. Run Chatbot:
   python src/chatbot.py
